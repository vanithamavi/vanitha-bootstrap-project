# vanitha-bootstrap-project
1. Objective Understanding

The goal of this task was to explore Bootstrap 5 components, remix them creatively, and build three fully responsive web pages: Home, About, and Contact.

2. Step-by-Step Process
Step 1: Setup
Created project structure (index.html, about.html, contact.html)
Included Bootstrap 5 via CDN
Initialized GitHub repository
Step 2: Component Exploration

I explored these sections in Bootstrap documentation:

Examples → Landing Page, Pricing, Dashboard
Components → Navbar, Cards, Forms, Carousel
Utilities → Spacing, Flexbox, Display

I extracted reusable UI patterns and remixed them instead of copy-pasting full examples.

Step 3: Page Design
Home Page: Built hero section, features grid, and a clean navbar/footer.
About Page: Added list-group, image section, and structured content.
Contact Page: Created a functional, responsive form layout.

Step 4: Responsiveness
Used Bootstrap container, row, col classes
Applied spacing utilities (p-3, my-5)
Tested layouts on mobile/tablet screen sizes

Step 5: Final Touches
Ensured visual consistency using a unified color palette
Added modern cards and shadow utilities
Clean indentation & semantic HTML

Step 6: Hosting
Pushed files to GitHub
Deployed using GitHub Pages

3. Tools Used
Bootstrap 5 Documentation
Bootstrap Examples
HTML5 / CSS3
Git & GitHub
ChatGPT (for guidance in structuring pages and understanding components)
4. Challenges Faced & Solutions
   
5.Challenge	Solution

Understanding Bootstrap grid	Practiced with container–row–col structure
Aligning spacing uniformly	Used Bootstrap utility classes instead of custom CSS
Creating unique layouts	Remixed components from multiple example pages
Deployment issues	Fixed folder path and reconfigured GitHub Pages

6. Learning Outcome

I gained hands-on experience in:
Bootstrap 5 layout & utilities
Component remixes instead of copying
Building responsive UI quickly
Clean, maintainable HTML structure
Hosting a live website with GitHub Pages

(including component exploration, designing pages, testing responsiveness, and deployment)
